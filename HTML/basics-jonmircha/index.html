Etiquetas semanticas estructurales de maquetacion HTML

Div
header - Cabecera del sitio web o seccion
main - define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o de una seccion
nav - Representa una navegaciom
article - Representa una seccion de autocontenid ( que por si sola se explica) tambien puede tener footer, header, section, content y demas
aside - Representa contenido complementario o secundario
section - Representa una seccion de contenido generico
address - Representa una direccion o una informacion de contacto

Etiquetas de Bloque VS Etiquetas de Línea

La etiqueta de bloque por exelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea (es decir enter) con sus etiquetas hermanas

La etiqueta de linea por exelencia es la span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra span
